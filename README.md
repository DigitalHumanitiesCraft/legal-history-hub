# Legal History Hub

Metadaten-Portal für die Forschungsprojekte der Abteilung II des [Max-Planck-Instituts für Rechtsgeschichte und Rechtstheorie](https://www.lhlt.mpg.de/).

## Komponenten

| Komponente | Beschreibung | Pfad |
|---|---|---|
| **Hub** | Discovery-Layer für Projektmetadaten | [`/`](index.html) |
| **Tutorial** | Lernressource für die eigenständige Arbeit am Hub | [`/tutorial/`](tutorial/) |

## Tech Stack

- **CMS:** Google Sheets + Sheets API
- **Frontend:** Vanilla JavaScript, Bootstrap 5
- **Tutorial:** Docsify (Markdown im Browser)
- **Hosting:** GitHub Pages

## Projekt

Ein Projekt von [Digital Humanities Craft](https://dhcraft.org) für das Max-Planck-Institut für Rechtsgeschichte und Rechtstheorie.
