# Workshop 2: Web-Grundlagen, Promptotyping & Claude Code

> Introduction to Web Development, Promptotyping, and Claude Code – der zweite Workshop im Legal History Hub Projekt.

**Datum:** 13.03.2026
**Dauer:** 4 Stunden (Remote)
**Leitung:** Christian Steiner

## Themen

- Setup für AI-gestütztes Coding (VS Code, Claude Code, Node.js, GitHub CLI)
- Wie funktioniert das Web? (HTTP, URL, Client/Server, Frontend/Backend)
- HTML als Markup Language (kein Programmieren)
- Git: Versionskontrolle, Commits, Branches, Pull Requests
- Vibe Coding vs. Informed Vibe Coding vs. Agentic Coding
- AI Coding Literacy: Computational Thinking, Requirement Engineering, Code Literacy
- Promptotyping: Die 4 Phasen (Preparation, Exploration, Distillation, Implementation)
- Promptotyping Documents (data.md, requirements.md, design.md)
- Context Window, Context Rot und Distillation
- Critical Expert in the Loop
- Hands-On: MoMA Collection Dataset (160.000+ Werke, data.md und requirements.md erstellen)
- Hands-On: Promptotyping in Practice (Prototyp generieren, reviewen, iterieren)
- Claude Code Survival Kit (CLAUDE.md, /clear, /compact, @filename)
- Claude Code Under the Hood (Plan Mode, Settings, Permissions, Skills)
